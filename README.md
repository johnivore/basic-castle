# basic-castle

Minimalist dotfiles for use with `homeshick`.


## Setup

Copy `bootstrap.sh` to the remote system and run it.

    wget https://gitlab.com/johnivore/basic-castle/raw/master/bootstrap.sh
    cat bootstrap.sh
    sh bootstrap.sh


## vim

    git clone https://github.com/vim-airline/vim-airline /home/john/.vim/bundle/vim-airline
    git clone https://github.com/vim-airline/vim-airline-themes /home/john/.vim/bundle/vim-airline-themes
    git clone https://github.com/vim-syntastic/syntastic.git /home/john/.vim/bundle/syntastic
    git clone https://github.com/lifepillar/vim-solarized8.git /home/john/.vim/pack/themes/opt/solarized8


## Android

If this is an Android system:

    homeshick clone https://gitlab.com/johnivore/android-castle.git
